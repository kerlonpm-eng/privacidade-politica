📜 Política de Privacidade - RxPython

Dados que Coletamos

A RxPython processa as mensagens enviadas em canais onde ela tem permissão de leitura. Para o funcionamento da IA, armazenamos temporariamente:

  • Conteúdo das mensagens: Apenas para gerar a resposta da IA.
  • IDs de Usuário e Servidor: Para o sistema de memória compartilhada funcionar e o bot saber com quem está falando.

Como Usamos os Dados

Os dados são usados exclusivamente para:

  • Responder comandos e interagir via chat.
  • Manter o contexto da conversa (Memória).
  • Melhorar as funcionalidades da RxPython.

Não vendemos ou compartilhamos dados de usuários para fins publicitários.

Provedores de Terceiros

As mensagens enviadas ao bot são processadas através da API do Groq e do Gemini (ou outros provedores configurados). Esses provedores possuem suas próprias políticas de privacidade sobre como processam dados de API.

Armazenamento e Exclusão

As informações de memória são vinculadas ao seu ID de usuário/servidor. Caso deseje que seus dados de memória sejam apagados, basta usar o comando !clear_memory.

Além disso, a memória compartilhada dura apenas 2 dias (ela é conectada ao canal em si, não ao servidor), e a memória de conversas com o usuário pode durar até 30 dias. Caso seja solicitado a algum administrador do bot, a memória pode ser excluída.
